Set as default tab

Safari

  • Choose Safari → Preferences, then click General
  • In the Homepage field, enter a webpage address: https://lifenometer.art

Chrome

  • At the top right, click Settings
  • Under Appearance, turn on Show Home button
  • Below Show Home button, type the address https://lifenometer.art

Edge, Brave, Ya.Browser, etc

  • The same as Chrome

Set as screensaver

Mac

  • Download WebViewScreenSaver and open the file
  • In the opened window choose WebViewScreenSaver on the left
  • Go to Screen Saver Options → Add URL. By default it will be Google, so click the Enter button and change the address to https://lifenometer.art/. Then click Tab button and put 86400 in the Seconds field.
  • Your new screen saver is set up!

About
